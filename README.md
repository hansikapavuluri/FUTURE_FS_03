# 🏫 School Management Website - MERN Stack

A full-stack school website built using the MERN stack (MongoDB, Express.js, React.js, Node.js) to provide an online platform for managing school information, admissions, and communication.

## 🚀 About The Project

This school management website is designed to showcase school details and simplify communication between students, parents, and the administration.

The project includes a modern responsive frontend, backend APIs, and database integration for managing school-related data.

## ✨ Features

- Home page with school information
- About school section
- Courses / Programs details
- Admissions enquiry form
- Contact form
- Student information management
- Admin dashboard
- CRUD operations
- Responsive design
- REST API integration
